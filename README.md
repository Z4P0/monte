# monte

base web project.
**requires node, grunt, and bower**

```
[monte]$ npm install
```

In a seperate tab run a Python server
```
$ python -m SimpleHTTPServer
$ python -m SimpleHTTPServer 8080  # optional port number argument
```

## Current Stack
- GreenSock (TweenMax)
- Howler
- Konami
- Zurb Foundation (jQuery/Fastclick/Modernizr)
